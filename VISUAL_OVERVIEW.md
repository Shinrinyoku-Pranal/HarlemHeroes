## 🎯 Navigation Flow

```
ENTRY POINTS:
├─ Direct URL → Home (Hero)
├─ "Listen Now" button → Listen (Audio)
├─ Navbar links → Any section
└─ Smooth scroll → Anchored sections

NAVIGATION PATH OPTIONS:
1. Linear (sequential): Home → About → Gallery → Listen → Script → Team
2. Direct (navbar): Jump to any section
3. Contextual (links): Follow CTAs and internal links
4. Gallery: Auto-advance or manual control

SMOOTH SCROLL TRIGGERS:
• All navbar links
• "Listen Now" CTA button
• Gallery pagination
• Scroll-to-top features
```
