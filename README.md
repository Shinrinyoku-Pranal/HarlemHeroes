# Harlem Heroes Podcast Website

## Overview
A beautiful, professionally designed website for the "Harlem Heroes" podcast—an engaging exploration of Langston Hughes, the Harlem Renaissance, jazz, blues, and their profound influence on American poetry and culture.

## Features

### 🎙️ Complete Podcast Experience
- **Hero Section** - Eye-catching landing page with stunning visual hierarchy
- **About Section** - Comprehensive overview of the podcast theme with key info cards
- **Audio Player** - Direct playback of the podcast episode with controls
- **Gallery** - Historical images from the Harlem Renaissance era featuring:
  - Langston Hughes portraits
  - Louis Armstrong
  - Bessie Smith
  - Duke Ellington
  - Count Basie
  - Historic Harlem Street scenes
  - Authentic 1920s-1930s photographs from Library of Congress and Wikimedia Commons

### 👥 Creator Spotlight
The website features detailed bios of the three creators:
- **Pranal Siripurapu** - Host & Producer
- **Emilia Striegel** - Researcher & Script Writer
- **Emma Buffington** - Producer & Audio Specialist

### 📚 Content Sections
1. **Featured Poetry** - Excerpt from "The Weary Blues" with analysis
2. **Episode Themes** - Four key theme cards exploring:
   - The Harlem Renaissance
   - Langston Hughes
   - Jazz & Blues Influence
   - Global Legacy

### ✨ Interactive Elements
- **Smooth Navigation** - Sticky navbar with smooth scroll functionality
- **Gallery Slideshow** - Auto-advancing image carousel with:
  - Next/Previous navigation buttons
  - Dot indicators
  - Keyboard arrow key controls
  - 8-second auto-advance
- **Scroll Animations** - Cards fade in as you scroll
- **Responsive Design** - Works seamlessly on mobile, tablet, and desktop

## Technical Stack

### HTML
- Semantic markup with proper structure
- Accessible navigation
- Embedded audio player with controls
- Optimized image gallery structure

### CSS
- **Color Scheme**: Dark theme (#1a1a1a) with gold accents (#d4a574)
- **Typography**: Georgia/Garamond serif fonts for elegance
- **Responsive Grid Layout**: CSS Grid and Flexbox
- **Animations**: Smooth transitions, fade effects, hover states
- **Mobile-First Design**: Fully responsive at all breakpoints

### JavaScript
- **Gallery Navigation**: 
  - `nextImage()` / `previousImage()` functions
  - `currentImage(n)` for direct navigation
  - `showImage(n)` for rendering logic
- **Auto-Advance**: 8-second interval with viewport detection
- **Keyboard Controls**: Arrow key navigation
- **Smooth Scrolling**: All navigation links
- **Intersection Observer**: Lazy animations on scroll
- **Active Link Tracking**: Dynamic navbar highlighting
- **Engagement Tracking**: Console logging for user interactions

## How to Use

1. **Open in Browser**: Simply open `index.html` in your web browser
2. **Navigate**: Use the sticky navigation bar to jump between sections
3. **Gallery**: Click Previous/Next buttons or use arrow keys to browse images
4. **Listen**: Click the play button on the audio player to listen to the full episode
5. **Responsive**: Works perfectly on all device sizes
